## Decisão central

A NOXUND **não deve lançar marketplace transacional no MVP**. O MVP correto é uma ferramenta fechada de inteligência de mercado, o **Hotspot Artists Report**, usada para validar se produtores de type beat mudam decisão de produção quando recebem sinais estruturados e verificáveis de tração recente.

Marketplace, sub-perfis de beatmaker, upload de beats, checkout, licenças, payouts e camada transacional entram somente depois que o Hotspot provar valor com produtores validados.
